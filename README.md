# personal-website
This is my personal website that I created to showcase my work and skills.

## stack:
- React (Vite)
- Tailwind CSS
- Framer Motion

## install and run:
1. Clone the repo
2. Run `yarn`
3. Run `yarn dev`

## deploy:
1. Run `firebase init`
2. Run `yarn build`
3. Run `firebase deploy`
